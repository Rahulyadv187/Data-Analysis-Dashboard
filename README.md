# Data-Analysis-Dashboard
An interactive E-Commerce Sales Analysis Dashboard built using Microsoft Excel to analyze business performance across sales, profit, orders, customers, regions, and product categories. The dashboard provides actionable insights through dynamic visualizations, KPI tracking, and filtering options.
